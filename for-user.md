# for-user.md

## Goal
Play published or playtest sessions through natural language actions.

## How to Play
1. Register/login as `user` (or use admin playtest session).
2. Start a game from the Player Journey panel.
3. Read scene narrative and use quick action chips or free text.
4. Submit action and review:
   - route type (avenue/wildcard/clarification)
   - explanation and confidence
   - updated points/turns
5. Follow stage transitions and ending grade once session completes.

## Tips
- Specific intent generally maps better than vague text.
- Clarification responses mean the system needs more detail.
- Wildcard routes are bounded by authored scene policy.
